# Roles del Sistema — Taller Municipal

##  Rol: admin
Acceso total al sistema.
Puede:
- Crear usuarios
- Crear roles
- Acceder a todos los módulos
- Ver todos los reportes
- Crear, editar y consultar todo

##  Rol: almacén
Acceso a:
- Inventario
- Recepciones
- Salidas
- Kardex

No puede:
- Crear órdenes de compra
- Crear solicitudes
- Ver reportes de auditoría

##  Rol: compras
Acceso a:
- Órdenes de compra
- Proveedores
- Diferencias OC
- Reportes de compras

No puede:
- Hacer salidas
- Hacer recepciones
- Ver inventario completo

##  Rol: mecánico
Acceso a:
- Crear solicitudes
- Ver sus órdenes de servicio
- Ver refacciones disponibles

No puede:
- Ver reportes
- Ver inventario
- Crear OC
- Hacer recepciones o salidas

##  Rol: auditor
Acceso a:
- Dashboard
- Reportes
- Kardex
- Consumo por OS
- Gasto por vehículo

No puede:
- Crear nada
- Modificar inventario
- Crear solicitudes
- Crear OC
